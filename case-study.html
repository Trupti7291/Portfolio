Case Study for myFlix full-stack project

Overview

  • myFlix is a web application, a client-side application developed in React, based on its existing server-side code using MERN stack.
  • The application consists of a list of all movies to the user.
  • Users can select a movie for more details(description, genre, director, image).
  • Users are able to create an account, update their personal data, and create a list of favorite movies.

    • Goal

  • To create a responsive web application to showcase my skills in full-stack (Front-end and Back-end) development in my professional portfolio.

    • Objective

  • The aim of the project was to have an ambitious full-stack project I can add to my professional portfolio.
  • Develop my skills to build the complete server-side and client-side for the application from scratch.

    • Back-end Programming


    The API is tested in Postman and includes user authentication and authorization code, data validation logic. It meets data security regulations. The API is deployed to Heroku.



    The API is a Node.js and Express application and uses REST architecture, with URL endpoints corresponding to the data operations. The database is built using MongoDB. The business logic is modeled with Mongoose. The API provides movie information in JSON format.
